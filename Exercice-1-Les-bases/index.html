Les bases de l'html

Les Balises

Les balises paires

Les balises HTML respectent une syntaxe simple et stricte :

  • Un chevron ouvrant (<)
  • le nom de la balise
  • des attributs (optionnels)
  • Un chevron fermant (>)

En général, les balises fonctionnent par paires. La première balise est une balise ouvrante et la seconde est fermante. Une balise fermante doit avoir le même nom que la balise ouvrante qui lui correspond. Une balise fermante doit contenir un slash (/) entre le chevron initial et le nom de la balise. Ainsi, si <p> est une balise ouvrante, </p> sera la balise fermante correspondante.

  • Les titres : <h1></h1>,<h2></h2>,<h3></h3>... etc
  • Les paragraphes : <p></p>
  • Les listes : <ul></ul>, <li></li>
  • Le formatage du texte : <s></s>, <b></b>
  • Les citations : <cite> </cite>
  • ...etc

Un élément HTML se compose d'une balise ouvrante, d'un contenu textuel et d'une balise fermante :

Les balises orphelines

Les balises orphelines sont des balises qui sont à la fois ouvrantes et fermante.Elles ne contiennent pas de contenu. On ferme donc la balise en ajoutant un slash (/) à la fin de la balise.Les balises AUTOFERMANTES sont soit des balises de type BLOC (exemple : <hr />), soit des balise EN LIGNE (exemple : <img />).

  • Les zones cliquables : <area>
  • Les sauts de ligne : <br>
  • Les images insérées : <img>

Les Attributs

Les élément HTML peuvent avoir un ou plusieurs attributs. Ce sont des valeurs supplémentaires qui permettent de configurer les éléments ou d'adapter leur comportement. L’élément a (pour "anchor") servant à créer des liens vers d’autres sites ou d’autres pages, va avoir besoin d’un attribut href qui va prendre comme valeur l'adresse de la page vers laquelle on souhaite faire un lien.

La Différence entre inline et block

Par défaut, les éléments HTML sont affichés selon l'un des modes suivants :

  • En-ligne / inline
    • N'occupe que la largeur indispensable à l'affichage du contenu et ne provoque pas de retour à la ligne. Exemple :
    • <img>
    • <strong></strong>
    • <span></span>
  • Bloc / block
    • Occupe toute la largeur disponible. Lorsque 2 éléments blocs se suivent dans une page, ils sont positionnés (par défaut) l'un sous l'autre. Exemple :
    • <p></p>
    • <article></article>
    • <header></header>

    Comparatif Markdown/HTML

    Insertion d'image

    • Markdown : ![NomImage](CheminAccesVersImage)
    • Html : <img src="CheminAccesVersImage">

    Insertion Citation

    Markdown :


    > Citation.
    >
    > -- <cite>Auteur</cite>

    Html :

    • Citation Longue : <blockquote></blockquote>
    • Citation Courte : <q></q>
    • Référence/Source : <cite></cite>
